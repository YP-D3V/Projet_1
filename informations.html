Formation Développement Web & Web Mobile

Description

Cette formation est destinée aux personnes débutantes passionnées par la technologie et à celles envisageant une carrière dans le développement informatique. Cours de pointe tels que la programmation orientée objet, le dev Back-End avec Java Spring Boot, Node.js, PHP et Symfony, sans oublier le Front-End avec HTML5, CSS et Javascript avec Angular ainsi qu’une multitude de projets pour mettre en pratique ses compétences.
